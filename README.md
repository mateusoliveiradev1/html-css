<div align="center">

- 🌟 **Dominar HTML5**: Estruturação semântica e moderna

- 🚀 **Boas Práticas**: Código limpo e profissional

## 📋 Pré-requisitos

<div align="center">

### 🎯 **O que você precisa saber antes de começar**

</div>

### ✅ **Conhecimentos Essenciais**

- 💻 **Noções básicas de informática**: Navegação em arquivos e pastas
- 🌐 **Conceitos de internet**: Como funcionam sites e navegadores
- 📝 **Editor de texto**: Familiaridade com qualquer editor (recomendado: VS Code)
- 🔍 **Curiosidade**: Vontade de aprender e experimentar!

### 🛠️ **Ferramentas Necessárias**

| Ferramenta | Descrição | Status |
|------------|-----------|--------|
| 🌐 **Navegador** | Chrome, Firefox, Edge ou Safari | ✅ Obrigatório |
| 📝 **Editor de Código** | VS Code (recomendado) | ✅ Obrigatório |
| 🔧 **Git** | Controle de versão | 🟡 Recomendado |
| 📱 **DevTools** | Ferramentas do desenvolvedor | 🟡 Recomendado |

### 📚 **Conhecimentos Opcionais (mas úteis)**

- 🎨 **Design básico**: Noções de cores, tipografia e layout
- 🖥️ **Lógica de programação**: Conceitos básicos ajudam no CSS
- 📐 **Matemática básica**: Para cálculos de layout e animações

## 🗺️ Roadmap de Estudos

<div align="center">

### 🎯 **Trilha de Aprendizado Recomendada**

</div>

<details>
<summary><strong>📈 Sequência Otimizada de Estudos</strong></summary>

### 🌱 **Fase 1: Fundamentos (Semanas 1-2)**
```
📚 Teoria Básica
├── 📖 Estrutura HTML
├── 🎨 Introdução ao CSS
└── 🔧 Configuração do ambiente

🏋️‍♂️ Prática
├── Ex001-Ex006: HTML básico
├── D001-D002: Primeiros desafios
└── Ex007-Ex009: Formatação de texto
```

### 🌿 **Fase 2: Estilização (Semanas 3-4)**
```
📚 CSS Intermediário
├── 🎨 Cores e fontes
├── 📦 Box Model
└── 🔗 Links e navegação

🏋️‍♂️ Prática
├── Ex010-Ex015: CSS externo
├── D003-D006: Desafios visuais
└── Ex016-Ex019: Cores e fontes
```

### 🚀 **Fase 3: Conceitos Avançados (Semanas 5-6)**
```
📚 CSS Avançado
├── 🖱️ Pseudo-classes
├── 📦 Modelo de caixas
└── 🖼️ Backgrounds e imagens

🏋️‍♂️ Prática
├── Ex020-Ex022: Interatividade
├── D007-D010: Projetos complexos
└── Ex023-Ex024: Tabelas e iframes
```

### 🏆 **Fase 4: Projetos Finais (Semanas 7-8)**
```
🎯 Projetos Completos
├── D011: Astronauta (Posicionamento)
├── D012: Cordel (Parallax)
├── D013-D014: Tabelas avançadas
└── 🔄 Revisão e consolidação
```

</details>

### ⏱️ **Cronograma Sugerido**

| Semana | Foco | Exercícios | Desafios | Tempo/Dia |
|--------|------|------------|----------|----------|
| 1-2 | 🌱 Fundamentos | Ex001-Ex009 | D001-D002 | 1-2h |

| 5-6 | 🚀 Avançado | Ex020-Ex024 | D007-D010 | 2-3h |

## 🔧 Troubleshooting

<details>
<summary><strong>📋 Problemas Comuns e Soluções</strong></summary>

### 🚫 **Problemas de Visualização**

**Problema:** Página não carrega ou aparece em branco
- ✅ Verifique se o arquivo `index.html` está na pasta correta
- ✅ Certifique-se de que todas as imagens estão no caminho correto
- ✅ Abra o DevTools (F12) e verifique se há erros no console

**Problema:** CSS não está sendo aplicado
- ✅ Verifique se o link para o arquivo CSS está correto no HTML
- ✅ Confirme se o arquivo CSS está salvo com a extensão `.css`
- ✅ Limpe o cache do navegador (Ctrl+F5)

### 🎨 **Problemas de Estilização**

**Problema:** Fontes não aparecem corretamente
- ✅ Verifique a conexão com a internet (Google Fonts)
- ✅ Confirme se o nome da fonte está escrito corretamente
- ✅ Adicione fontes de fallback: `font-family: 'Roboto', Arial, sans-serif;`

**Problema:** Layout quebrado ou elementos sobrepostos
- ✅ Verifique se há conflitos no CSS
- ✅ Use o DevTools para inspecionar elementos
- ✅ Confirme se as propriedades `display`, `position` e `float` estão corretas

### 📱 **Problemas de Responsividade**

**Problema:** Site não funciona bem no mobile
- ✅ Adicione a meta tag viewport: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- ✅ Use unidades relativas (%, em, rem) em vez de pixels fixos
- ✅ Teste em diferentes tamanhos de tela

### 🔗 **Problemas com Links e Navegação**

**Problema:** Links não funcionam
- ✅ Verifique se o caminho do arquivo está correto
- ✅ Confirme se a extensão do arquivo está incluída (.html)
- ✅ Use caminhos relativos para arquivos locais

### 💡 **Dicas Gerais**

- 🔍 **Sempre use o DevTools** para debugar problemas
- 💾 **Salve os arquivos** antes de testar no navegador
- 🔄 **Atualize a página** após fazer alterações
- 📝 **Valide seu código** em [validator.w3.org](https://validator.w3.org/)
- 🎯 **Teste em diferentes navegadores** (Chrome, Firefox, Safari)

</details>
