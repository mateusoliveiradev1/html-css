🎯 Desafio 01: Mensagens

Desafio prático para testar seus conhecimentos sobre parágrafos e quebras de linha

🎯 Objetivos de Aprendizagem

📝

Uso Correto de Parágrafos

Aplicar a tag <p> de forma adequada para estruturar o conteúdo textual do desafio.

↩️

Quebras de Linha Estratégicas

Utilizar <br> para criar padrões visuais específicos com asteriscos e formatação.

🎨

Padrões Visuais

Criar um design visual simples usando apenas HTML e caracteres especiais como asteriscos.

🧩

Resolução de Problemas

Desenvolver habilidades de resolução de problemas aplicando conceitos HTML na prática.

🎯 Descrição do Desafio

Neste desafio, você deve criar um padrão visual usando asteriscos (*) e quebras de linha. O objetivo é praticar:

  • ▶ Uso correto da tag <p>
  • ▶ Aplicação da tag <br> para quebras de linha
  • ▶ Estruturação de conteúdo em HTML
  • ▶ Criação de padrões visuais simples

📋 Requisitos do Desafio

📈 Padrão Crescente

Crie um parágrafo com asteriscos em ordem crescente:

* Um
** Dois
*** Três
**** Quatro

📉 Padrão Decrescente

Crie outro parágrafo com asteriscos em ordem decrescente:

**** Quatro
*** Três
** Dois
* Um

✅ Resultado Esperado

Desafio das Mensagens

* Um
** Dois
*** Três
**** Quatro

**** Quatro
*** Três
** Dois
* Um

💡 Solução do Desafio

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mensagens</title>
</head>
<body>
    <h1>Desafio das Mensagens</h1>
    <p>
        * Um<br>
        ** Dois<br>
        *** Três<br>
        **** Quatro
    </p>

    <p>
        **** Quatro<br>
        *** Três<br>
        ** Dois<br>
        * Um
    </p>
</body>
</html>

🔍 Análise da Solução

📝 Estrutura de Parágrafos

Uso de dois parágrafos distintos (<p>) para separar logicamente os dois padrões, aproveitando o espaçamento automático do navegador.

↩️ Quebras de Linha

Aplicação estratégica da tag <br> para criar quebras de linha dentro de cada parágrafo, mantendo a coesão visual.

🎨 Padrões Visuais

Criação de um efeito visual simples usando apenas texto e estrutura HTML, demonstrando como o HTML pode criar layouts básicos.

🏷️ Semântica HTML

Uso apropriado das tags semânticas: <h1> para o título principal e <p> para agrupar conteúdo relacionado.

🚀 Variações e Melhorias

🔣 Variação 1: Outros símbolos

Tente criar o mesmo padrão usando outros símbolos como +, -, #, ou emojis.

📏 Variação 2: Padrão maior

Estenda o padrão para ir até 6 ou 8 níveis, criando uma pirâmide maior.

🎯 Variação 3: Padrão centralizado

Pesquise como usar espaços (&nbsp;) para centralizar visualmente o padrão.

🔄 Variação 4: Múltiplos padrões

Crie uma página com vários padrões diferentes: crescente, decrescente, losango, etc.
