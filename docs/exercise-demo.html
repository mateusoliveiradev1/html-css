Estrutura HTML Básica

Todo documento HTML deve seguir uma estrutura padrão. Vamos criar nosso primeiro arquivo:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Esta é minha primeira página HTML.</p>
</body>
</html>

Elementos Importantes

  • <!DOCTYPE html> - Declara o tipo de documento
  • <html> - Elemento raiz da página
  • <head> - Contém metadados da página
  • <body> - Contém o conteúdo visível
  • <title> - Define o título da aba do navegador
