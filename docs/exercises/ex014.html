🎬 Exercício 014

Áudio e Vídeo HTML5

🎵 Sobre este Exercício

Neste décimo quarto exercício, você aprenderá como incorporar áudio e vídeo em suas páginas web usando HTML5. Os elementos <audio> e <video> oferecem controle nativo sobre mídia sem necessidade de plugins.

Vamos explorar diferentes formatos, atributos de controle, acessibilidade e técnicas avançadas para uma experiência multimídia rica.

🎵 Elemento Audio

O elemento <audio> permite incorporar conteúdo de áudio diretamente na página:

🎧Áudio Básico

Controles padrão do navegador

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Seu navegador não suporta áudio.
</audio>

🔄Áudio com Autoplay

Reprodução automática (muted)

<audio controls autoplay muted loop>
  <source src="background.mp3" type="audio/mpeg">
  <source src="background.ogg" type="audio/ogg">
  Fallback text
</audio>

🎬 Elemento Video

O elemento <video> permite incorporar conteúdo de vídeo com controles nativos:

📹Vídeo Básico

Com poster e controles

<video controls poster="poster.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <track kind="subtitles" src="pt-br.vtt"          srclang="pt-br" label="Português">
  Seu navegador não suporta vídeo.
</video>

🔇Vídeo Silencioso

Autoplay permitido quando muted

<video controls autoplay muted loop>
  <source src="demo.mp4" type="video/mp4">
  <source src="demo.webm" type="video/webm">
  Fallback content
</video>

⚙️ Atributos Importantes

Os elementos de mídia HTML5 oferecem diversos atributos para controlar o comportamento:

controls

Exibe os controles padrão do navegador (play, pause, volume, etc.)

autoplay

Inicia a reprodução automaticamente. Requer muted na maioria dos navegadores.

loop

Reproduz a mídia em loop contínuo quando chega ao fim.

muted

Inicia a mídia sem som. Necessário para autoplay funcionar.

preload

Controla o pré-carregamento: none, metadata, auto

poster (vídeo)

Imagem exibida antes da reprodução do vídeo começar.

width/height

Define as dimensões do player de vídeo em pixels.

crossorigin

Controla requisições CORS: anonymous, use-credentials

♿ Acessibilidade e Legendas

Para tornar o conteúdo multimídia acessível, use o elemento <track> para fornecer legendas, descrições e capítulos:

<video controls>
  <source src="video.mp4" type="video/mp4">
  
  <!-- Legendas em português -->
  <track kind="subtitles" src="pt-br.vtt"          srclang="pt-br" label="Português" default>
  
  <!-- Legendas em inglês -->
  <track kind="subtitles" src="en.vtt"          srclang="en" label="English">
  
  <!-- Descrições para deficientes visuais -->
  <track kind="descriptions" src="descriptions.vtt"          srclang="pt-br" label="Descrições">
  
  <!-- Capítulos para navegação -->
  <track kind="chapters" src="chapters.vtt"          srclang="pt-br" label="Capítulos">
  
  Seu navegador não suporta vídeo.
</video>

Tipos de track:

  • subtitles - Legendas traduzidas
  • captions - Legendas para surdos (incluem efeitos sonoros)
  • descriptions - Descrições de áudio para deficientes visuais
  • chapters - Títulos de capítulos para navegação
  • metadata - Metadados não visíveis ao usuário
