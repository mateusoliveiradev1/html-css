Design Responsivo Avançado

Técnicas modernas de responsividade

Elemento Responsivo
/* Container queries (futuro) */
@container (min-width: 400px) {
    .card {
        display: flex;
        flex-direction: row;
    }
}

/* Clamp para tipografia fluida */
.title {
    font-size: clamp(1.5rem, 4vw, 3rem);
}

/* Media queries modernas */
@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
    }
}

Boas Práticas de CSS

Diretrizes para código CSS profissional

  • Organize com metodologias: Use BEM, SMACSS ou Atomic CSS para estruturar seu código
  • Variáveis CSS: Centralize valores em custom properties para facilitar manutenção
  • Mobile First: Comece o design pelos dispositivos móveis e expanda para desktop
  • Performance: Minimize reflows, use transform/opacity para animações
  • Acessibilidade: Respeite prefers-reduced-motion e mantenha contraste adequado
  • Semântica: Use seletores semânticos e evite !important desnecessário
  • Modularidade: Crie componentes reutilizáveis e independentes
  • Documentação: Comente código complexo e mantenha style guides

Variáveis CSS e Design Tokens

Sistema de design escalável

/* Design System com CSS Variables */
:root {
    /* Cores semânticas */
    --color-primary: #2563eb;
    --color-secondary: #64748b;
    --color-success: #10b981;
    --color-warning: #f59e0b;
    --color-error: #ef4444;
    
    /* Espaçamento */
    --space-xs: 0.25rem;
    --space-sm: 0.5rem;
    --space-md: 1rem;
    --space-lg: 1.5rem;
    --space-xl: 2rem;
    
    /* Tipografia */
    --font-size-sm: 0.875rem;
    --font-size-base: 1rem;
    --font-size-lg: 1.125rem;
    --font-size-xl: 1.25rem;
    
    /* Sombras */
    --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
    --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1);
    --shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1);
}

/* Tema escuro */
@media (prefers-color-scheme: dark) {
    :root {
        --color-background: #1e293b;
        --color-text: #f1f5f9;
    }
}
