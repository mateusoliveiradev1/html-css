🖼️ Exercício 009

Imagens em HTML

📸 Sobre este Exercício

Neste nono exercício, você aprenderá como trabalhar com imagens em HTML. As imagens são elementos fundamentais para criar páginas web atrativas e informativas.

Vamos explorar diferentes formatos de imagem, atributos importantes, responsividade e boas práticas de acessibilidade.

⚙️ Atributos Importantes

A tag <img> possui vários atributos essenciais:

<!-- Atributos obrigatórios -->
<img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

<!-- Dimensões -->
<img src="foto.jpg" alt="Foto" width="300" height="200">

<!-- Título e informações extras -->
<img src="logo.png" alt="Logo da empresa" title="Nossa marca">

<!-- Carregamento otimizado -->
<img src="grande.jpg" alt="Imagem grande" loading="lazy">

<!-- Múltiplas fontes -->
<img src="imagem.jpg" alt="Descrição" srcset="pequena.jpg 300w, media.jpg 600w, grande.jpg 1200w">

📱 Imagens Responsivas

Criar imagens que se adaptam a diferentes tamanhos de tela:

CSS para Responsividade:

/* Imagem responsiva básica */
img {
  max-width: 100%;
  height: auto;
}

/* Imagem que ocupa toda a largura */
.full-width {
  width: 100%;
  height: auto;
}

/* Imagem com largura fixa máxima */
.limited-width {
  max-width: 500px;
  width: 100%;
  height: auto;
}

🎯 Elemento Picture

O elemento <picture> permite servir diferentes imagens baseadas em condições específicas:

<picture>
  <!-- Para telas pequenas -->
  <source media="(max-width: 600px)" srcset="mobile.jpg">

  <!-- Para telas médias -->
  <source media="(max-width: 1200px)" srcset="tablet.jpg">

  <!-- Para formatos modernos -->
  <source type="image/webp" srcset="imagem.webp">

  <!-- Fallback -->
  <img src="desktop.jpg" alt="Imagem responsiva">
</picture>

Vantagens do elemento picture:

  • Diferentes imagens para diferentes tamanhos de tela
  • Suporte a formatos modernos com fallback
  • Melhor performance e experiência do usuário
  • Controle total sobre qual imagem carregar

♿ Acessibilidade em Imagens

  • Sempre use o atributo alt: Descreva o conteúdo da imagem para leitores de tela
  • Alt vazio para imagens decorativas: Use alt="" para imagens puramente decorativas
  • Seja descritivo: "Gato preto dormindo no sofá" é melhor que "gato"
  • Evite "imagem de" ou "foto de": Leitores de tela já identificam que é uma imagem
  • Use title para informações extras: Tooltips com contexto adicional
  • Considere o contraste: Texto sobre imagens deve ter bom contraste
  • Teste com leitores de tela: Verifique se a experiência faz sentido

⚡ Otimização de Imagens

Dicas para melhorar a performance das suas imagens:

<!-- Carregamento lazy -->
<img src="imagem.jpg" alt="Descrição" loading="lazy">

<!-- Múltiplas resoluções -->
<img src="imagem.jpg"
     alt="Descrição"
     srcset="pequena.jpg 300w,
             media.jpg 600w,
             grande.jpg 1200w"
     sizes="(max-width: 600px) 300px,
            (max-width: 1200px) 600px,
            1200px">

<!-- Preload para imagens importantes -->
<link rel="preload" as="image" href="hero-image.jpg">

Boas práticas de otimização:

  • Comprima imagens antes de usar
  • Use o formato adequado para cada tipo de imagem
  • Implemente lazy loading para imagens abaixo da dobra
  • Use srcset para servir diferentes resoluções
  • Considere usar WebP ou AVIF para navegadores modernos
  • Defina dimensões para evitar layout shift

🎯 Exemplo Completo

Vamos criar uma galeria de imagens responsiva:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Galeria de Fotos</title>
  <style>
    .gallery {
      display: grid;
      grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
      gap: 20px;
      padding: 20px;
    }
    .gallery img {
      width: 100%;
      height: 200px;
      object-fit: cover;
      border-radius: 8px;
      transition: transform 0.3s ease;
    }
    .gallery img:hover {
      transform: scale(1.05);
    }
  </style>
</head>
<body>
  <h1>Minha Galeria</h1>
  <div class="gallery">
    <img src="foto1.jpg" alt="Paisagem montanhosa ao pôr do sol" loading="lazy">
    <img src="foto2.jpg" alt="Praia com águas cristalinas" loading="lazy">
    <img src="foto3.jpg" alt="Floresta verde exuberante" loading="lazy">
    <img src="foto4.jpg" alt="Cidade iluminada à noite" loading="lazy">
    <img src="foto5.jpg" alt="Campo de flores coloridas" loading="lazy">
    <img src="foto6.jpg" alt="Cachoeira em meio à natureza" loading="lazy">
  </div>
</body>
</html>
