📊 Exercício 011

Tabelas em HTML

📋 Sobre este Exercício

Neste décimo primeiro exercício, você aprenderá como criar e estruturar tabelas em HTML. As tabelas são fundamentais para apresentar dados organizados de forma clara e acessível.

Vamos explorar a estrutura semântica das tabelas, estilização, responsividade e boas práticas de acessibilidade.

🏗️ Estrutura Básica de uma Tabela

Uma tabela HTML bem estruturada utiliza elementos semânticos específicos:

<table>
  <caption>Título da Tabela</caption>
  <thead>
    <tr>
      <th>Cabeçalho 1</th>
      <th>Cabeçalho 2</th>
      <th>Cabeçalho 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Dados 1</td>
      <td>Dados 2</td>
      <td>Dados 3</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">Rodapé da tabela</td>
    </tr>
  </tfoot>
</table>

🧩 Elementos das Tabelas

Cada elemento tem sua função específica na estrutura da tabela:

📊

<table>

Função: Container principal

Uso: Envolve toda a tabela

Atributos: border, cellpadding, cellspacing

📝

<caption>

Função: Título da tabela

Uso: Descreve o conteúdo

Posição: Primeiro elemento após <table>

🎯

<thead>

Função: Cabeçalho da tabela

Uso: Agrupa linhas de cabeçalho

Conteúdo: Elementos <th>

📄

<tbody>

Função: Corpo da tabela

Uso: Agrupa dados principais

Conteúdo: Linhas com <td>

🔚

<tfoot>

Função: Rodapé da tabela

Uso: Totais, resumos

Posição: Pode vir antes do <tbody>

📏

<tr>

Função: Linha da tabela

Uso: Agrupa células horizontalmente

Conteúdo: <th> ou <td>

🏷️

<th>

Função: Célula de cabeçalho

Uso: Títulos de colunas/linhas

Estilo: Negrito e centralizado

📦

<td>

Função: Célula de dados

Uso: Conteúdo da tabela

Atributos: colspan, rowspan

⚙️ Atributos Importantes

As tabelas possuem vários atributos para melhorar funcionalidade e acessibilidade:

<!-- Mesclagem de células -->
<td colspan="2">Ocupa 2 colunas</td>
<td rowspan="3">Ocupa 3 linhas</td>

<!-- Cabeçalhos com escopo -->
<th scope="col">Cabeçalho de coluna</th>
<th scope="row">Cabeçalho de linha</th>

<!-- Associação de cabeçalhos -->
<th id="nome">Nome</th>
<td headers="nome">João Silva</td>

<!-- Agrupamento de colunas -->
<colgroup>
  <col style="background-color: #f0f0f0;">
  <col span="2" style="background-color: #e0e0e0;">
</colgroup>

📱 Tabelas Responsivas

Estratégias para tornar tabelas responsivas em dispositivos móveis:

1. Scroll Horizontal:

.table-container {
  overflow-x: auto;
}

.table-container table {
  min-width: 600px;
}

2. Empilhamento em Mobile:

@media (max-width: 768px) {
  table, thead, tbody, th, td, tr {
    display: block;
  }

  thead tr {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }

  tr {
    border: 1px solid #ccc;
    margin-bottom: 10px;
  }

  td {
    border: none;
    position: relative;
    padding-left: 50%;
  }

  td:before {
    content: attr(data-label) ": ";
    position: absolute;
    left: 6px;
    width: 45%;
    font-weight: bold;
  }
}

♿ Acessibilidade em Tabelas

  • Use <caption> sempre: Descreve o propósito da tabela para leitores de tela
  • Defina scope nos cabeçalhos: Use scope="col" ou scope="row" em elementos <th>
  • Use headers para células complexas: Associe células a múltiplos cabeçalhos
  • Estruture semanticamente: Use <thead>, <tbody> e <tfoot> adequadamente
  • Evite tabelas para layout: Use apenas para dados tabulares
  • Forneça resumos: Use summary ou descrições para tabelas complexas
  • Teste com leitores de tela: Verifique se a navegação faz sentido

🎨 Estilização Avançada

CSS moderno para criar tabelas atrativas e funcionais:

/* Tabela moderna */
table {
  border-collapse: collapse;
  width: 100%;
  background: white;
  border-radius: 10px;
  overflow: hidden;
  box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

/* Cabeçalho estilizado */
th {
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: white;
  padding: 15px;
  text-align: left;
  font-weight: 600;
}

/* Células com hover */
td {
  padding: 12px 15px;
  border-bottom: 1px solid #f0f0f0;
  transition: background-color 0.3s ease;
}

/* Efeito zebra */
tbody tr:nth-child(even) {
  background-color: rgba(0,0,0,0.02);
}

/* Hover nas linhas */
tbody tr:hover {
  background-color: rgba(102,126,234,0.05);
}

/* Responsividade */
@media (max-width: 768px) {
  .table-container {
    overflow-x: auto;
  }
  
  table {
    min-width: 600px;
  }
}
