Exercício 023 - CSS Animações e Transições

Aprenda a criar animações fluidas e transições elegantes com CSS para dar vida às suas páginas web

🎯 Objetivo do Exercício

Neste exercício, você aprenderá a implementar animações e transições CSS para criar interfaces mais dinâmicas e envolventes. Exploraremos propriedades como transform, transition, animation e keyframes para dar movimento aos elementos da página.

🎨 Transições CSS

Aprenda a criar transições suaves entre estados de elementos usando a propriedade transition. Controle duração, timing e propriedades afetadas.

🎭 Animações CSS

Domine o uso de @keyframes e animation para criar animações complexas e personalizadas que executam automaticamente.

🔄 Transformações

Explore as propriedades transform para rotacionar, escalar, mover e distorcer elementos de forma dinâmica.

Código de Exemplo

Estrutura básica para animações CSS

/* Transição simples */
.elemento {
    transition: all 0.3s ease;
}

.elemento:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 20px rgba(0,0,0,0.1);
}

/* Animação com keyframes */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.animado {
    animation: fadeIn 0.6s ease-out;
}
