🔤 Tipos de Input HTML5

HTML5 introduziu diversos tipos de input que oferecem validação nativa e interfaces otimizadas para diferentes dispositivos:

type="email"

Validação automática de formato de email

type="tel"

Teclado numérico em dispositivos móveis

type="url"

Validação de formato de URL

type="number"

Apenas números, com controles de incremento

type="range"

Slider para seleção de valores

type="date"

Seletor de data nativo

type="time"

Seletor de horário

type="color"

Seletor de cores nativo

✅ Validação HTML5

HTML5 oferece validação nativa através de atributos e pseudo-classes CSS:

<!-- Atributos de validação -->
<input type="email" required>
<input type="text" minlength="3" maxlength="20">
<input type="number" min="1" max="100">
<input type="tel" pattern="\([0-9]{2}\) [0-9]{4,5}-[0-9]{4}">

<!-- CSS para estados de validação -->
<style>
  input:valid { border-color: green; }
  input:invalid { border-color: red; }
  input:required { border-left: 3px solid blue; }
</style>

♿ Acessibilidade em Formulários

Boas práticas para tornar formulários acessíveis a todos os usuários:

<!-- Labels associados corretamente -->
<label for="username">Nome de usuário</label>
<input type="text" id="username" name="username">

<!-- Fieldset para agrupar campos relacionados -->
<fieldset>
  <legend>Informações de contato</legend>
  <input type="email" name="email">
  <input type="tel" name="phone">
</fieldset>

<!-- ARIA para melhor acessibilidade -->
<input type="password" aria-describedby="pwd-help">
<div id="pwd-help">Mínimo 8 caracteres</div>

<!-- Indicação de campos obrigatórios -->
<input type="text" required aria-required="true">
