📋 Exercício 007

Listas em HTML

📝 Sobre este Exercício

Neste sétimo exercício, você aprenderá como criar diferentes tipos de listas em HTML. As listas são fundamentais para organizar informações de forma estruturada e hierárquica.

Vamos explorar listas ordenadas, não ordenadas e listas de definição, cada uma com suas características e usos específicos.

📋 Tipos de Listas

HTML oferece três tipos principais de listas, cada uma adequada para diferentes situações:

📌 Lista Não Ordenada

<ul> - Unordered List

Para itens sem ordem específica

Marcadores: •, ○, ■

🔢 Lista Ordenada

<ol> - Ordered List

Para itens com sequência lógica

Numeração: 1, 2, 3... ou A, B, C...

📖 Lista de Definição

<dl> - Definition List

Para termos e suas definições

Estrutura: termo + descrição

💻 Exemplos Práticos

Vamos ver como cada tipo de lista funciona na prática:

📌 Lista Não Ordenada

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Resultado:

  • HTML
  • CSS
  • JavaScript

🔢 Lista Ordenada

<ol>
  <li>Aprender HTML</li>
  <li>Estudar CSS</li>
  <li>Praticar JavaScript</li>
</ol>

Resultado:

  1. Aprender HTML
  2. Estudar CSS
  3. Praticar JavaScript

📖 Lista de Definição

<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação</dd>
  <dt>CSS</dt>
  <dd>Folhas de estilo</dd>
</dl>

Resultado:

HTML
Linguagem de marcação
CSS
Folhas de estilo

🔗 Listas Aninhadas

Você pode criar listas dentro de outras listas para organizar informações hierarquicamente:

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Backend
    <ul>
      <li>Node.js</li>
      <li>Python</li>
    </ul>
  </li>
</ul>

⚙️ Atributos Especiais

As listas ordenadas possuem atributos que permitem personalizar a numeração:

type="1"

Numeração decimal (padrão)

Exemplo: 1, 2, 3, 4...

type="A"

Letras maiúsculas

Exemplo: A, B, C, D...

type="a"

Letras minúsculas

Exemplo: a, b, c, d...

type="I"

Números romanos maiúsculos

Exemplo: I, II, III, IV...

type="i"

Números romanos minúsculos

Exemplo: i, ii, iii, iv...

start="5"

Inicia a numeração em 5

Exemplo: 5, 6, 7, 8...

🎯 Exemplo Completo

Vamos criar uma página com diferentes tipos de listas:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Minhas Listas</title>
</head>
<body>
  <h1>Guia de Estudos</h1>

  <h2>Tecnologias para Aprender</h2>
  <ul>
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript</li>
    <li>React</li>
  </ul>

  <h2>Plano de Estudos</h2>
  <ol>
    <li>Dominar HTML básico</li>
    <li>Aprender CSS fundamentals</li>
    <li>Estudar JavaScript</li>
    <li>Construir projetos</li>
  </ol>

  <h2>Glossário</h2>
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
</body>
</html>

📋 Boas Práticas

Use o tipo certo: Escolha o tipo de lista baseado no conteúdo - ordenada para sequências, não ordenada para coleções.

Mantenha consistência: Use o mesmo estilo de lista para conteúdos similares em toda a página.

Evite listas muito longas: Considere dividir listas extensas em seções menores.

Use CSS para estilização: Prefira CSS para personalizar a aparência das listas em vez de atributos HTML.
