CSS Transitions

Transitions criam animações suaves entre estados de elementos:

/* Sintaxe básica das transitions */ .elemento { /* Propriedade a animar | Duração | Timing function | Delay */ transition: opacity 0.3s ease 0.1s; /* Múltiplas propriedades */ transition: opacity 0.3s ease, transform 0.5s cubic-bezier(0.68, -0.55, 0.265, 1.55); /* Shorthand para todas as propriedades */ transition: all 0.3s ease; } /* Estados diferentes */ .elemento:hover { opacity: 0.7; transform: scale(1.1); } /* Propriedades individuais */ .elemento { transition-property: transform, opacity; transition-duration: 0.3s, 0.5s; transition-timing-function: ease-out, ease-in; transition-delay: 0s, 0.1s; }

CSS Animations e Keyframes

Animations permitem criar sequências complexas de animação:

Bounce Animation

@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-20px); } 60% { transform: translateY(-10px); } } .bounce { animation: bounce 2s infinite; }

Pulse Animation

@keyframes pulse { 0% { transform: scale(1); box-shadow: 0 0 0 0 rgba(255, 0, 0, 0.7); } 70% { transform: scale(1.05); box-shadow: 0 0 0 10px rgba(255, 0, 0, 0); } 100% { transform: scale(1); box-shadow: 0 0 0 0 rgba(255, 0, 0, 0); } } .pulse { animation: pulse 2s infinite; }

Fade In Up

@keyframes fadeInUp { from { opacity: 0; transform: translateY(30px); } to { opacity: 1; transform: translateY(0); } } .fade-in-up { animation: fadeInUp 0.6s ease-out; }

Rotate Scale

@keyframes rotateScale { 0% { transform: rotate(0deg) scale(1); } 50% { transform: rotate(180deg) scale(1.2); } 100% { transform: rotate(360deg) scale(1); } } .rotate-scale { animation: rotateScale 2s ease-in-out infinite; }

Propriedades de Animation

/* Propriedades completas de animation */ .elemento { /* Nome da animação */ animation-name: minha-animacao; /* Duração */ animation-duration: 2s; /* Timing function */ animation-timing-function: ease-in-out; /* Delay */ animation-delay: 0.5s; /* Número de repetições */ animation-iteration-count: infinite; /* ou número específico */ /* Direção */ animation-direction: alternate; /* normal, reverse, alternate, alternate-reverse */ /* Fill mode */ animation-fill-mode: forwards; /* none, forwards, backwards, both */ /* Play state */ animation-play-state: running; /* running, paused */ /* Shorthand */ animation: minha-animacao 2s ease-in-out 0.5s infinite alternate forwards; } /* Múltiplas animações */ .elemento { animation: fadeIn 1s ease-out, slideUp 0.8s ease-out 0.2s, bounce 2s ease-in-out 1s infinite; }

Timing Functions

Controle a velocidade e o ritmo das animações:

/* Timing functions predefinidas */ .linear { animation-timing-function: linear; } .ease { animation-timing-function: ease; } .ease-in { animation-timing-function: ease-in; } .ease-out { animation-timing-function: ease-out; } .ease-in-out { animation-timing-function: ease-in-out; } /* Cubic-bezier personalizada */ .bounce-effect { animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); } .smooth { animation-timing-function: cubic-bezier(0.25, 0.46, 0.45, 0.94); } /* Steps para animações discretas */ .typewriter { animation-timing-function: steps(10, end); } /* Exemplos de cubic-bezier populares */ .ease-out-back { animation-timing-function: cubic-bezier(0.175, 0.885, 0.32, 1.275); } .ease-in-back { animation-timing-function: cubic-bezier(0.6, -0.28, 0.735, 0.045); } .ease-in-out-back { animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); }

Dicas de Performance

  • Use transform e opacity para animações mais fluidas
  • Evite animar propriedades que causam reflow (width, height, margin)
  • Use will-change para otimizar elementos que serão animados
  • Prefira transform: translateZ(0) para ativar aceleração de hardware
  • Use animation-fill-mode: forwards para manter o estado final
  • Teste performance em dispositivos móveis
  • Use prefers-reduced-motion para acessibilidade
  • Mantenha animações entre 200ms-500ms para boa UX
