Exercício 02: Parágrafos e Quebras de Linha

Aprendendo a estruturar texto com parágrafos, quebras de linha e símbolos especiais em HTML

Código do Exercício

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Parágrafos e Quebras</title>
</head>
<body>
    <h1>Parágrafos e quebras de linha</h1>
    <hr>
    <p>Você pode escrever um parágrafo de qualquer jeito.
    Basta colocar tudo no meio do par de tags &lt;p&gt; e &lt;/p&gt;</p>
    <p>Se precisar quebrar o texto em algum lugar
    específico <br>como essa linha aqui,
    você pode usar a tag &lt;br&gt;.</p>
    <p>Vamos adicionar alguns símbolos especiais:
    &reg; &copy; &trade; &euro; &pound; &yen; &cent;
    &Delta; &uarr; &uarr;</p>
    <p>Vamos adicionar alguns emojis:
    &#x1F596; &#x1F913;</p>
</body>
</html>

Resultado Final

Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

Se precisar quebrar o texto em algum lugar específico
como essa linha aqui, você pode usar a tag <br>.

Vamos adicionar alguns símbolos especiais: ® © ™ € £ ¥ ¢ Δ ↑ ↑

Vamos adicionar alguns emojis: 🖖 🤓

📚 O que você está aprendendo

📝 Parágrafos

A tag <p> define um parágrafo. O navegador adiciona automaticamente espaço antes e depois de cada parágrafo.

↩️ Quebras de Linha

A tag <br> força uma quebra de linha sem criar um novo parágrafo. É uma tag vazia (self-closing).

🔣 Entidades HTML

Códigos especiais como &copy; para © e &lt; para < permitem exibir caracteres reservados.

😀 Emojis Unicode

Use códigos como &#x1F596; para 🖖 ou &#x1F913; para 🤓 e adicione emojis ao seu HTML.

🔤 Principais Entidades HTML

Símbolos Básicos

&lt; → <
&gt; → >
&amp; → &
&quot; → "
&apos; → '

Direitos Autorais

&copy; → ©
&reg; → ®
&trade; → ™

Moedas

&euro; → €
&pound; → £
&yen; → ¥
&cent; → ¢

Setas e Símbolos

&uarr; → ↑
&darr; → ↓
&larr; → ←
&rarr; → →
&Delta; → Δ

🚀 Desafios Práticos

📝 Desafio 1: Crie um poema

Escreva um poema de 4 versos usando parágrafos e quebras de linha. Use <br> para separar os versos dentro de cada estrofe.

🔣 Desafio 2: Use entidades HTML

Crie um parágrafo explicando como usar as tags <p> e <br> (use as entidades para mostrar as tags literalmente).

😀 Desafio 3: Adicione emojis

Adicione pelo menos 5 emojis diferentes usando códigos Unicode. Pesquise os códigos de seus emojis favoritos!
