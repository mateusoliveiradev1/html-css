📋 Exercício 013

Listas em HTML

📝 Sobre este Exercício

Neste décimo terceiro exercício, você aprenderá como criar e estilizar diferentes tipos de listas em HTML. As listas são fundamentais para organizar informações de forma estruturada e hierárquica.

Vamos explorar listas ordenadas, não ordenadas, de definição e técnicas avançadas de estilização com CSS.

📚 Tipos de Listas

HTML oferece três tipos principais de listas, cada uma com seu propósito específico:

🔸Lista Não Ordenada (ul)

Uso: Itens sem ordem específica

  • Primeiro item
  • Segundo item
  • Terceiro item
  • Quarto item
<ul>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ul>

🔢Lista Ordenada (ol)

Uso: Itens com ordem específica

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo
  4. Quarto passo
<ol>
  <li>Primeiro passo</li>
  <li>Segundo passo</li>
  <li>Terceiro passo</li>
</ol>

📖Lista de Definição (dl)

Uso: Termos e suas definições

HTML
Linguagem de marcação para estruturar conteúdo web
CSS
Linguagem de estilo para apresentação visual
JavaScript
Linguagem de programação para interatividade
<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação</dd>
  <dt>CSS</dt>
  <dd>Linguagem de estilo</dd>
</dl>

🎯 Tipos de Marcadores

CSS oferece diversos tipos de marcadores para personalizar suas listas:

list-style-type: disc
  • Item com disco
  • Outro item
  • Mais um item
list-style-type: circle
  • Item com círculo
  • Outro item
  • Mais um item
list-style-type: square
  • Item com quadrado
  • Outro item
  • Mais um item
list-style-type: decimal
  1. Primeiro item
  2. Segundo item
  3. Terceiro item
list-style-type: lower-alpha
  1. Item a
  2. Item b
  3. Item c
list-style-type: upper-roman
  1. Item I
  2. Item II
  3. Item III

🎨 Estilização Avançada

CSS moderno permite criar listas altamente personalizadas:

/* Lista personalizada com ícones */
.custom-list {
  list-style: none;
  padding: 0;
}

.custom-list li {
  position: relative;
  padding-left: 30px;
  margin-bottom: 10px;
}

.custom-list li::before {
  content: "✓";
  position: absolute;
  left: 0;
  color: #48bb78;
  font-weight: bold;
}

/* Lista numerada personalizada */
.custom-numbered {
  counter-reset: item;
  list-style: none;
}

.custom-numbered li {
  counter-increment: item;
}

.custom-numbered li::before {
  content: counter(item);
  background: #667eea;
  color: white;
  border-radius: 50%;
  width: 25px;
  height: 25px;
  display: inline-flex;
  align-items: center;
  justify-content: center;
  margin-right: 10px;
}

/* Responsividade */
@media (max-width: 768px) {
  .list-grid {
    grid-template-columns: 1fr;
  }
}
