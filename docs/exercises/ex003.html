Exercício 003

Imagens em HTML - Caminhos, formatos e boas práticas

📚 Tipo

Exercício Básico

⏱️ Duração

25-40 minutos

🎯 Dificuldade

Iniciante

🛠️ Tecnologias

HTML5

🎯 Objetivo

Neste exercício, você vai aprender a trabalhar com imagens em HTML, explorando diferentes tipos de caminhos, formatos de imagem e boas práticas para otimização e acessibilidade.

O que você vai aprender:

  • Como inserir imagens com a tag <img>
  • Diferença entre caminhos relativos e absolutos
  • Como usar URLs externas para imagens
  • Importância do atributo alt para acessibilidade
  • Organização de arquivos em pastas
  • Diferentes formatos de imagem (PNG, JPG, WebP)

💻 Código Completo

📄 index.html 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Teste com imagems</title>
</head>
<body>
    <h1>Testando carga e iamgems</h1>
    <p>Abaixo voce vai ver uma imagem que esta na mesma pasta</p>
    <img src="logo-html.png" alt="Logo HTML 5">
    
    <p>Podemos tambem carregar imagems que estao em sub-pastas</p>
    <img src="imagems/logo-css.png" alt="Logo CSS3">
    
    <p>Tambem podemos carregar imagems externas</p>
    <img src="https://cdn.iconscout.com/icon/free/png-256/free-javascript-23-1174949.png" alt="Logo JavaScript">
    
    <p>Teste PUBG</p>
    <img src="pubg-400.webp" alt="PUBG Game">
    
    <p>Teste BLEACH ICHIGO</p>
    <img src="Bleach-200.jpg" alt="Bleach Ichigo">
</body>
</html>

📁 Tipos de Caminhos

🏠 Caminho Relativo - Mesma Pasta

Quando a imagem está na mesma pasta do arquivo HTML:

src="logo-html.png"

Vantagem: Simples e direto

📂 Caminho Relativo - Subpasta

Quando a imagem está em uma pasta dentro do projeto:

src="imagems/logo-css.png"

Vantagem: Organização melhor dos arquivos

🌐 URL Externa

Quando a imagem está hospedada em outro servidor:

src="https://exemplo.com/imagem.png"

Cuidado: Dependência de servidor externo

🖼️ Formatos de Imagem

📸 JPG/JPEG

Ideal para fotografias e imagens com muitas cores.

  • Boa compressão
  • Não suporta transparência
  • Pode perder qualidade

🎨 PNG

Ideal para logos, ícones e imagens com transparência.

  • Suporta transparência
  • Sem perda de qualidade
  • Arquivos maiores

⚡ WebP

Formato moderno com excelente compressão.

  • Menor tamanho
  • Suporta transparência
  • Nem todos os navegadores suportam

♿ Acessibilidade com Alt

O atributo alt é essencial para acessibilidade:

  • Leitores de tela: Descrevem a imagem para pessoas com deficiência visual
  • Conexão lenta: Texto aparece enquanto a imagem carrega
  • SEO: Motores de busca entendem o conteúdo da imagem
  • Erro de carregamento: Texto alternativo é exibido

✅ Boas práticas para Alt:

  • Seja descritivo mas conciso
  • Não use "imagem de" ou "foto de"
  • Para imagens decorativas, use alt=""
  • Descreva o conteúdo, não a aparência

📋 Organização de Arquivos

Uma boa estrutura de pastas facilita a manutenção:

├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/ (ou img/)
│   ├── logos/
│   │   ├── logo-html.png
│   │   └── logo-css.png
│   ├── photos/
│   │   └── hero-image.jpg
│   └── icons/
│       └── favicon.ico
└── docs/
    └── readme.md

🧠 Conceitos Importantes

🔄 Responsividade

Use CSS para tornar imagens responsivas:

img {
    max-width: 100%;
    height: auto;
}

⚡ Performance

  • Otimize o tamanho das imagens
  • Use formatos modernos (WebP)
  • Considere lazy loading para muitas imagens
  • Comprima imagens antes de usar

🔒 Segurança

  • Cuidado com URLs externas
  • Verifique a origem das imagens
  • Considere hospedar imagens importantes localmente

🚀 Desafios Extras

  1. Crie uma galeria com 6 imagens diferentes
  2. Organize as imagens em pastas por categoria
  3. Adicione descrições alt significativas para cada imagem
  4. Experimente diferentes formatos (JPG, PNG, WebP)
  5. Crie uma página "Sobre" com sua foto e biografia
  6. Use uma imagem externa como plano de fundo (com CSS)
