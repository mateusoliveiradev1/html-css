📦 Flexbox - Layout Flexível

O Flexbox é uma ferramenta poderosa para criar layouts flexíveis e responsivos com facilidade:

.flex-container {
  display: flex;
  justify-content: center; /* Alinhamento horizontal */
  align-items: center; /* Alinhamento vertical */
  gap: 20px; /* Espaçamento entre itens */
}

.flex-item {
  flex: 1; /* Cresce igualmente */
  flex-basis: 200px; /* Tamanho base */
}

🏗️ CSS Grid - Layout em Grade

CSS Grid oferece controle bidimensional completo sobre layouts, permitindo criar estruturas complexas com facilidade:

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 colunas iguais */
  grid-template-rows: 100px 200px; /* 2 linhas fixas */
  gap: 20px; /* Espaçamento */
}

.grid-item {
  grid-column: span 2; /* Ocupa 2 colunas */
  grid-row: 1 / 3; /* Da linha 1 à 3 */
}

✨ Animações e Transições

CSS permite criar animações fluidas e interações envolventes sem JavaScript:

/* Transição simples */
.elemento {
  transition: all 0.3s ease;
}

.elemento:hover {
  transform: scale(1.1);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

/* Animação com keyframes */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

.bounce {
  animation: bounce 2s infinite;
}

📱 Design Responsivo Avançado

Técnicas modernas para criar layouts que se adaptam perfeitamente a qualquer dispositivo:

/* Mobile First */
.container {
  padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 2rem;
    max-width: 1200px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    padding: 3rem;
  }
}

/* Grid responsivo */
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
}

💡 Casos de Uso Práticos

Flexbox é ideal para:

  • Navegação horizontal
  • Centralização de elementos
  • Distribuição de espaço
  • Layouts de componentes

CSS Grid é perfeito para:

  • Layouts de página completos
  • Galerias de imagens
  • Dashboards complexos
  • Estruturas bidimensionais

Animações melhoram:

  • Feedback visual
  • Experiência do usuário
  • Transições suaves
  • Interatividade
