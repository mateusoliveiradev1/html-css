CSS Tipografia

📝 Sobre este Exercício

Neste décimo sétimo exercício, você dominará a arte da tipografia em CSS. Aprenderá sobre famílias de fontes, pesos, tamanhos, espaçamento e como criar hierarquias visuais eficazes.

A tipografia é fundamental para a comunicação visual e a experiência do usuário. Vamos explorar todas as propriedades que tornam o texto legível, atraente e funcional.

🔤 Famílias de Fontes

CSS oferece diferentes tipos de famílias de fontes, cada uma com características específicas para diferentes contextos:

/* Serif - com serifas, elegante para títulos */
font-family: 'Playfair Display', serif;

/* Sans-serif - sem serifas, moderna e legível */
font-family: 'Inter', sans-serif;

/* Monospace - espaçamento fixo, ideal para código */
font-family: 'Fira Code', monospace;

/* Fallback stack completo */
font-family: 'Inter', 'Helvetica Neue', Arial, sans-serif;

🎭 Serif - Elegante e Clássica

"A tipografia é a arte e técnica de arranjar tipos de forma a tornar a linguagem escrita legível, legível e atraente quando exibida."

Uso: Títulos, textos longos, design editorial

🔧 Sans-serif - Moderna e Limpa

A tipografia sans-serif é caracterizada pela ausência de serifas (pequenos traços decorativos). É amplamente utilizada em interfaces digitais devido à sua excelente legibilidade em telas.

Uso: Interfaces, textos de corpo, navegação

💻 Monospace - Espaçamento Fixo

function calculateTypography() {
  const fontSize = '16px';
  const lineHeight = 1.6;
  return { fontSize, lineHeight };
}

Uso: Código, dados tabulares, terminais

💪 Peso das Fontes (Font Weight)

O peso da fonte determina a espessura dos caracteres, criando hierarquia e ênfase visual:

/* Valores numéricos (100-900) */
font-weight: 100; /* Thin */
font-weight: 300; /* Light */
font-weight: 400; /* Normal/Regular */
font-weight: 500; /* Medium */
font-weight: 600; /* Semi-bold */
font-weight: 700; /* Bold */
font-weight: 900; /* Black */

/* Valores nomeados */
font-weight: normal; /* 400 */
font-weight: bold; /* 700 */
font-weight: lighter; /* Relativo ao pai */
font-weight: bolder; /* Relativo ao pai */

📏 Tamanho das Fontes (Font Size)

O tamanho da fonte é crucial para legibilidade e hierarquia visual. Use unidades relativas para melhor responsividade:

/* Unidades absolutas */
font-size: 16px; /* Pixels - fixo */
font-size: 12pt; /* Points - impressão */

/* Unidades relativas (recomendado) */
font-size: 1rem; /* Relativo ao root */
font-size: 1.2em; /* Relativo ao pai */
font-size: 100%; /* Percentual */

/* Unidades de viewport */
font-size: 4vw; /* 4% da largura da tela */
font-size: 2vh; /* 2% da altura da tela */

📏 Altura da Linha (Line Height)

A altura da linha afeta diretamente a legibilidade e o ritmo visual do texto. É o espaço vertical entre as linhas:

/* Valores sem unidade (recomendado) */
line-height: 1.2; /* Apertado - títulos */
line-height: 1.5; /* Normal - texto base */
line-height: 1.8; /* Relaxado - leitura longa */

/* Valores com unidade */
line-height: 24px; /* Absoluto */
line-height: 1.5em; /* Relativo ao font-size */
line-height: 150%; /* Percentual */

📖 Comparação de Line Heights

Line Height: 1.2 (Apertado)

Este texto tem line-height apertado (1.2). É ideal para títulos e cabeçalhos onde queremos economia de espaço vertical. Não é recomendado para textos longos pois pode prejudicar a legibilidade.

Line Height: 1.5 (Normal)

Este texto tem line-height normal (1.5). É o padrão recomendado para a maioria dos textos de corpo. Oferece boa legibilidade sem ocupar espaço excessivo na página.

Line Height: 1.8 (Relaxado)

Este texto tem line-height relaxado (1.8). É excelente para textos longos e leitura prolongada, pois o espaço extra entre as linhas reduz a fadiga visual.

Line Height: 2.0 (Muito Relaxado)

Este texto tem line-height muito relaxado (2.0). Usado em contextos especiais onde se deseja muito espaço entre linhas, como em textos poéticos ou designs minimalistas.

🔤 Espaçamento entre Letras (Letter Spacing)

O letter-spacing controla o espaço horizontal entre caracteres, afetando a densidade e o ritmo visual do texto:

/* Valores em em (recomendado) */
letter-spacing: -0.05em; /* Apertado */
letter-spacing: 0; /* Normal */
letter-spacing: 0.1em; /* Espaçado */
letter-spacing: 0.2em; /* Muito espaçado */

/* Valores em pixels */
letter-spacing: -1px; /* Apertado */
letter-spacing: 2px; /* Espaçado */

🔄 Transformação de Texto

A propriedade text-transform permite alterar a capitalização do texto sem modificar o HTML:

/* Transformações de texto */
text-transform: uppercase; /* MAIÚSCULAS */
text-transform: lowercase; /* minúsculas */
text-transform: capitalize; /* Primeira Letra Maiúscula */
text-transform: none; /* Sem transformação */

✨ Decoração de Texto

A propriedade text-decoration adiciona linhas decorativas ao texto:

/* Decorações de texto */
text-decoration: underline; /* Sublinhado */
text-decoration: overline; /* Linha superior */
text-decoration: line-through; /* Riscado */
text-decoration: none; /* Sem decoração */

/* Decoração personalizada */
text-decoration: underline wavy red; /* Sublinhado ondulado vermelho */

📐 Alinhamento de Texto

A propriedade text-align controla o alinhamento horizontal do texto dentro de seu contêiner:

/* Alinhamentos de texto */
text-align: left; /* Alinhado à esquerda */
text-align: center; /* Centralizado */
text-align: right; /* Alinhado à direita */
text-align: justify; /* Justificado */

🎯 Tipos de Alinhamento

Alinhado à Esquerda: Este texto está alinhado à esquerda. É o padrão para textos em português e a maioria dos idiomas ocidentais. Oferece boa legibilidade para textos longos.
Centralizado: Este texto está centralizado. Ideal para títulos, citações curtas e elementos que precisam de destaque visual. Não recomendado para textos longos.
Alinhado à Direita: Este texto está alinhado à direita. Usado em contextos específicos como assinaturas, datas ou quando se quer criar um efeito visual particular.
Justificado: Este texto está justificado, ou seja, alinhado tanto à esquerda quanto à direita. O espaçamento entre palavras é ajustado para que todas as linhas tenham a mesma largura. Comum em textos impressos e publicações.

🏗️ Hierarquia Tipográfica

Uma boa hierarquia tipográfica guia o leitor através do conteúdo, criando ordem visual e facilitando a compreensão:

Título Principal (H1)

Este é um parágrafo de destaque (lead) que introduz o conteúdo principal. Geralmente é maior que o texto normal e serve como ponte entre o título e o corpo do texto.

Título Secundário (H2)

Este é um parágrafo normal que compõe o corpo do texto. Deve ter boa legibilidade e espaçamento adequado para leitura confortável. O tamanho base recomendado é 16px ou 1rem.

Subtítulo (H3)

Os subtítulos ajudam a organizar o conteúdo em seções menores, facilitando a navegação e compreensão do texto.

Subtítulo Menor (H4)

Usado para subdivisões ainda menores do conteúdo, mantendo a hierarquia visual clara.

Este é um texto pequeno, usado para informações secundárias, notas de rodapé ou detalhes menos importantes.

💡 Dicas Práticas de Tipografia

🎯 Legibilidade:

  • Use no máximo 2-3 famílias de fontes por projeto
  • Mantenha line-height entre 1.4-1.6 para textos de corpo
  • Evite textos muito longos em uma única linha (45-75 caracteres)
  • Garanta contraste adequado entre texto e fundo

🎨 Hierarquia Visual:

  • Use escala tipográfica consistente (1.2x, 1.5x, 2x)
  • Combine peso e tamanho para criar contraste
  • Mantenha espaçamento consistente entre elementos
  • Use cores para reforçar a hierarquia

📱 Responsividade:

  • Use unidades relativas (rem, em) em vez de pixels
  • Ajuste tamanhos para diferentes dispositivos
  • Teste legibilidade em telas pequenas
  • Considere orientação do dispositivo
