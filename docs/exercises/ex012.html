📝 Exercício 012

Formulários em HTML

📋 Sobre este Exercício

Neste décimo segundo exercício, você aprenderá como criar formulários interativos em HTML. Os formulários são essenciais para coletar dados dos usuários de forma organizada e acessível.

Vamos explorar diferentes tipos de campos, validação, acessibilidade e boas práticas para criar formulários eficientes.

🏗️ Estrutura Básica de um Formulário

Um formulário HTML bem estruturado utiliza elementos semânticos específicos:

<form action="/processar" method="post">
  <fieldset>
    <legend>Informações Pessoais</legend>
    
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    
    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Enviar</button>
  </fieldset>
</form>

🧩 Tipos de Campos

HTML5 oferece diversos tipos de campos para diferentes necessidades:

📝

text

Uso: Texto simples

Exemplo: Nome, título

<input type="text" name="nome">
📧

email

Uso: Endereços de e-mail

Validação: Automática

<input type="email" name="email">
🔒

password

Uso: Senhas

Segurança: Texto oculto

<input type="password" name="senha">
🔢

number

Uso: Números

Controles: min, max, step

<input type="number" min="1" max="100">
📅

date

Uso: Datas

Interface: Calendário

<input type="date" name="nascimento">
📞

tel

Uso: Telefones

Mobile: Teclado numérico

<input type="tel" name="telefone">
🌐

url

Uso: URLs

Validação: Formato de URL

<input type="url" name="website">
🎨

color

Uso: Cores

Interface: Seletor de cor

<input type="color" name="cor">
📄

file

Uso: Upload de arquivos

Atributos: accept, multiple

<input type="file" accept=".pdf,.jpg">

💻 Exemplo Prático

Vamos criar um formulário completo de cadastro:

Informações Pessoais
Preferências
<form action="/cadastro" method="post">
  <fieldset>
    <legend>Informações Pessoais</legend>
    
    <label for="nome">Nome Completo *</label>
    <input type="text" id="nome" name="nome" required>
    
    <label for="email">E-mail *</label>
    <input type="email" id="email" name="email" required>
    
    <label for="telefone">Telefone</label>
    <input type="tel" id="telefone" name="telefone">
  </fieldset>
  
  <fieldset>
    <legend>Preferências</legend>
    
    <label for="genero">Gênero</label>
    <select id="genero" name="genero">
      <option value="">Selecione...</option>
      <option value="masculino">Masculino</option>
      <option value="feminino">Feminino</option>
    </select>
    
    <input type="checkbox" id="termos" name="termos" required>
    <label for="termos">Aceito os termos</label>
  </fieldset>
  
  <button type="submit">Cadastrar</button>
</form>

✅ Validação de Formulários

  • required: Torna o campo obrigatório
  • pattern: Define uma expressão regular para validação
  • min/max: Define valores mínimos e máximos para números e datas
  • minlength/maxlength: Define tamanho mínimo e máximo para texto
  • step: Define incrementos para campos numéricos
  • placeholder: Fornece dicas visuais para o usuário
  • autocomplete: Melhora a experiência com preenchimento automático
  • novalidate: Desabilita validação HTML5 (para validação customizada)

♿ Acessibilidade em Formulários

Formulários acessíveis são fundamentais para uma web inclusiva:

<!-- Labels associados corretamente -->
<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

<!-- Agrupamento com fieldset -->
<fieldset>
  <legend>Informações de Contato</legend>
  <!-- campos relacionados -->
</fieldset>

<!-- Mensagens de erro -->
<input type="email" aria-describedby="email-error">
<div id="email-error" role="alert">E-mail inválido</div>

<!-- Campos obrigatórios -->
<label for="email">E-mail <span aria-label="obrigatório">*</span></label>
<input type="email" id="email" required aria-required="true">

🎨 Estilização Avançada

CSS moderno para criar formulários atrativas e funcionais:

/* Estilo base para inputs */
input, select, textarea {
  width: 100%;
  padding: 12px 15px;
  border: 2px solid #e2e8f0;
  border-radius: 8px;
  font-size: 1rem;
  transition: all 0.3s ease;
}

/* Estado de foco */
input:focus, select:focus, textarea:focus {
  outline: none;
  border-color: #667eea;
  box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.1);
}

/* Estados de validação */
input:valid {
  border-color: #48bb78;
}

input:invalid:not(:placeholder-shown) {
  border-color: #f56565;
}

/* Botões estilizados */
button[type="submit"] {
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: white;
  padding: 15px 30px;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: transform 0.2s ease;
}

button[type="submit"]:hover {
  transform: translateY(-2px);
}

/* Responsividade */
@media (max-width: 768px) {
  .form-row {
    grid-template-columns: 1fr;
  }
}
