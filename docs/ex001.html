Exercício 001

Primeiro HTML - Criando sua primeira página web

📚 Tipo

Exercício Básico

⏱️ Duração

15-30 minutos

🎯 Dificuldade

Iniciante

🛠️ Tecnologias

HTML5

🎯 Objetivo

Este é o seu primeiro contato com HTML! Neste exercício, você vai aprender a criar uma página web básica usando a estrutura fundamental do HTML5. É o primeiro passo para se tornar um desenvolvedor web.

O que você vai aprender:

  • Estrutura básica de um documento HTML
  • Como usar as tags essenciais: <html>, <head>, <body>
  • Configurar metadados importantes como charset e viewport
  • Criar títulos e parágrafos
  • Usar elementos de separação como <hr>

💻 Código Completo

📄 index.html 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu primeiro exercício</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <hr>
    <p>Estou praticando HTML e estou muito feliz!</p>
    <hr>
</body>
</html>

📖 Explicação Detalhada

🏗️ Estrutura do Documento

  • <!DOCTYPE html> - Declara que este é um documento HTML5
  • <html lang="pt-br"> - Elemento raiz que define o idioma da página
  • <head> - Contém metadados que não são exibidos na página
  • <body> - Contém todo o conteúdo visível da página

🔧 Metadados Importantes

  • charset="UTF-8" - Define a codificação de caracteres para suportar acentos
  • viewport - Configura a visualização em dispositivos móveis
  • <title> - Define o título que aparece na aba do navegador

📝 Elementos de Conteúdo

  • <h1> - Título principal da página (maior hierarquia)
  • <p> - Parágrafo de texto
  • <hr> - Linha horizontal para separar conteúdo

🧠 Conceitos Fundamentais

🏷️ Tags HTML

As tags são os blocos de construção do HTML. Elas são escritas entre colchetes angulares < > e geralmente vêm em pares: uma tag de abertura e uma de fechamento.

🌐 Semântica

Cada tag tem um significado específico. O <h1> representa o título mais importante, enquanto <p> representa um parágrafo de texto.

📱 Responsividade

A meta tag viewport é essencial para que sua página funcione bem em dispositivos móveis.

🚀 Desafios Extras

  1. Adicione mais parágrafos com diferentes conteúdos
  2. Experimente usar outros níveis de títulos (h2, h3, h4)
  3. Adicione uma lista não ordenada com seus hobbies
  4. Inclua um link para seu site favorito
  5. Mude o título da página para algo mais personalizado
